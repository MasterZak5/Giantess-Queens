# Giantess Queens
Giantess Queens mod for Crusader Kings 3

This mod is an open source project contributed to by many over the years.
If you have an interest in contributing, join the discord server and share your improvements.

Giantess Queens is a macrophilia mod that adds Giantesses to the world of Crusader Kings 3. You can play as Giantesses themselves or as the tiny humans that live under their rule, or stand against them.
